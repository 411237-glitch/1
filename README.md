# 作業備忘錄

這是一個使用HTML5製作的作業備忘錄應用程式。

## 功能

- **週曆視圖**: 顯示一週的日曆，可以調整年份和月份，導航到上一週或下一週。
- **作業設定**: 選擇科目和輸入頁數（單頁或範圍），同一科目可設定多次。
- **視覺化**: 每個科目有不同顏色，在日曆中以百分比條顯示當天作業。
- **點數系統**: 完成作業獲得隨機孢子，頁數越多獲得越多孢子。
- **種植系統**: 使用隨機孢子種植蕈菇，獎品依稀有度機率不同。
- **收集系統**: 收集種植中的蕈菇，重複的可以換成隨機孢子。點擊菇類查看詳細描述。
- **獎池查看**: 查看所有獎池項目、機率和兌換隨機孢子數量。

## 科目顏色
- 國文: 淺紅
- 數學: 淺綠
- 英文: 淺藍
- 地理: 淺黃
- 歷史: 淺粉
- 公民: 淺青
- 地科: 淺橙
- 生物: 淺綠黃
- 化學: 淺紫
- 物理: 淺粉紅
- 其他: 灰色

## 蕈菇種類
- **常見**: 金針菇, 香菇, 杏鮑菇, 鴻喜菇, 木耳 (機率高，兌換1孢子)
- **稀有**: 猴頭菇, 舞菇, 竹笙, 雞油菌 (機率中等，兌換3孢子)
- **史詩**: 松茸, 牛肝菌, 羊肚菌, 虎掌菇 (機率低，兌換7孢子)
- **傳說**: 黑松露, 白松露, 野生冬蟲夏草, 巨型蟻巢傘 (機率極低，兌換15孢子)

## 使用方法

1. 打開index.html在瀏覽器中。
2. 使用控制按鈕調整年份、月份和週。
3. 點選日曆中的日期。
4. 選擇科目和輸入頁數，新增作業。
5. 完成作業後點擊"完成"按鈕獲得隨機孢子（根據頁數）。
6. 如需刪除作業，點擊"刪除"按鈕並確認。
7. 使用隨機孢子點擊"種植"按鈕種植蕈菇。
8. 點擊"查看獎池"查看所有項目。
9. 在收集區點擊蕈菇查看描述。
